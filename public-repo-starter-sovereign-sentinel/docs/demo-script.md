# Demo Script

1. Open UiPath Automation Cloud and the Sovereign Sentinel case.
2. Submit `sample-events/mutation-proposal.json`.
3. Show the case classification stage.
4. Show the Mutation Review Agent summary.
5. Show the Risk & Safety Sentinel Agent result.
6. Show the Consensus Verification Agent result.
7. Route the evidence packet to a human reviewer.
8. Choose approve, pause, or rollback.
9. Show the final audit and lineage summary.

Core message:

AI-Fi Sovereign Sentinel does not make autonomous agents less powerful. It makes them governable.
